## ZMQ Chat App

#### Team Members: Suhaas Yerramreddy, Varun Arora, Anvit Gupta

## Functionality ##

* Encrypted single user chat
* Encrypted group chat
* In case of user disconnect, recovering all single user and group chat data for that user

## Techstack ##

* Python
* ZeroMQ
* Amazon EC2
* Influxdb
* Flask
